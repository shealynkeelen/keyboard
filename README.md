# keyboard
This is for Circuits Design class. It is a keyboard that executes different functionalities supporting Mac.

# Wirless Version
To run the wireless version, makes sure you have the mac_listener.py file saved somewhere on your mac. Make sure that you install the library in the terminal by typing "pip3 install flask pyautogui". Save the file and run it in your mac terminal by direct it oot where it is locating, and type "python3 mac_listener.py"
